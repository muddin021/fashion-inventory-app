# Fashion Inventory Management Application

A command-line inventory management system for tracking fashion items, built with Python.

## Features
- Add, view, update, and remove inventory items
- Duplicate item detection — automatically updates quantity if item already exists
- Input validation and error handling across all user inputs
- Persistent JSON storage
- Total inventory value calculator

## Tech Stack
- Python 3
- JSON

## How to Run
1. Clone the repo
2. Run `python inventory_app.py`
